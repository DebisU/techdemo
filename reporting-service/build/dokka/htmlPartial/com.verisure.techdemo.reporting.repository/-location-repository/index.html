LocationRepository

@Repository
interface LocationRepository : JpaRepository<LocationEntity, Long>

Spring Data JPA Repository for Location Events.

Provides database access methods for LocationEntity with custom JPQL queries for aggregated reporting.

Inherited Methods

From JpaRepository:

  • save(entity): Persist a location event

  • count(): Total number of events

  • findAll(): All events (use with caution)

  • deleteAll(): Clear all events

Query Performance

The getAggregatedStats query performs grouping across all records. For large datasets (>1M rows), consider:

  • Materialized views

  • Pre-computed aggregates

  • Time-bounded queries
