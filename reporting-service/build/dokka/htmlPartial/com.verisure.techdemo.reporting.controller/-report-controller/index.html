ReportController

@RestController
@RequestMapping(value = ["/api/reports"])
class ReportController(reportService: ReportService)

REST Controller for Location Reports.

Exposes HTTP endpoints for querying aggregated location analytics. Results are cached via ReportService for improved performance.

Endpoints

MethodPathDescription
GET/api/reportsGlobal report (cached)
GET/api/reports/{id}User-specific report
GET/api/reports/healthHealth check endpoint

Example Usage

# Get global report
curl http://localhost:8082/api/reports

# Get user-specific report
curl http://localhost:8082/api/reports/user123

  • 404 Not Found: User has no location data
