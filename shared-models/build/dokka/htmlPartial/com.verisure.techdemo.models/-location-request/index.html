LocationRequest

data class LocationRequest(val userId: String, val latitude: Double, val longitude: Double, val timestamp: Instant = Instant.now())

Data Transfer Object for Incoming Location Requests.

Represents the HTTP request body for location submissions. Includes built-in validation via Kotlin's init block.

JSON Example

{
  "userId": "user-12345",
  "latitude": 40.4168,
  "longitude": -3.7038,
  "timestamp": "2024-01-15T10:30:00Z"
}
Content copied to clipboard

Validation Rules

  • userId: Must not be blank

  • latitude: Must be in range -90, 90

  • longitude: Must be in range -180, 180

  • timestamp: Optional, defaults to current time

Usage

@PostMapping
fun submitLocation(@RequestBody request: LocationRequest): ResponseEntity<...> {
    // Validation happens automatically on construction
    val event = LocationEvent.from(request)
    // ...
}
