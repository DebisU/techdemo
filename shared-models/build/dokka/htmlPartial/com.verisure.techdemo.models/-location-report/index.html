data class LocationReport(val userId: String, val totalLocations: Long, val lastLatitude: Double, val lastLongitude: Double, val lastUpdated: Instant)

Report DTO for Individual User Location Summary.

Contains aggregated location statistics for a single user. Used in both global reports and user-specific queries.
