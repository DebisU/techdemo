ReportResponse

data class ReportResponse(val totalUsers: Long, val totalEvents: Long, val reports: List<LocationReport>, val generatedAt: Instant = Instant.now())

Aggregated Report Response DTO.

Top-level response for the global report endpoint containing:

  • System-wide statistics (total users, total events)

  • Per-user location summaries

  • Generation timestamp

JSON Example

{
  "totalUsers": 25,
  "totalEvents": 5000,
  "reports": [...],
  "generatedAt": "2024-01-15T10:30:00Z"
}

Caching

This response is cached in Redis with a 60-second TTL. The generatedAt field helps clients understand data freshness.
